# 🚚 S.R Logistics Texas Dashboard (Power BI)

## 📊 Overview

A powerful, data-driven dashboard designed to optimize logistics operations across Texas. This project focuses on tracking performance, identifying inefficiencies, and improving overall delivery experience using interactive Power BI visualizations.

## 🚀 Key Highlights

- ✨ Tracked 1,130+ monthly orders across multiple hubs
- 🏢 Managed operations over 6 hubs
- 👨‍✈️ Monitored performance of 55 drivers
- 🚛 Analyzed utilization of 45 vehicles
